# COSS.io

Very simple, not even close to fully finished [COSS.io](https://coss.io/) API Wrapper.


#### Installation

* Clone repository
* Install dependencies

```
$ npm install

or

$ yarn install
```

* Build the API

```
$ npm run build

or

$ yarn build
```

* Start the example app

```
$ npm run start

or

$ yarn start
```
